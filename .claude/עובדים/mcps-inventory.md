## כללי חיבור MCP

1. **לעולם לא לשמור API keys בקבצי הפרויקט** — להשתמש ב-environment variables
2. **לבדוק שהמפתח עובד** לפני שמוסיפים לעובד
3. **לעדכן את הפרופיל של העובד** אחרי חיבור MCP
4. **לעדכן קובץ זה** אחרי כל שינוי
