## שלב 2 — תרגום לקוד (מצב בנייה לוקאלית)

במידה ונבחרה בנייה ישירה של קובץ הקוד:
1. **שימוש ב-Tailwind CDN (כדי שהקובץ יעבוד מיידית):**
   ```html
   <script src="https://cdn.tailwindcss.com"></script>
   ```
2. **הטמעת ה-Design Tokens (צבעים ופונטים) ב-Config של Tailwind:**
   הגדר את הצבעים והפונטים של ריבה כמשתני Tailwind בתוך תגית הסקריפט:
   ```html
   <script>
     tailwind.config = {
       theme: {
         extend: {
           colors: {
             primary: '#...',
             secondary: '#...',
             background: '#...',
           },
           fontFamily: {
             sans: ['Inter', 'sans-serif'], // או הפונט העברי שנבחר
           }
         }
       }
     }
   </script>
   ```
3. **בניית שלד ה-HTML והמחלקות הרספונסיביות:**
   - בנה את המבנה של עידו.
   - השתמש במחלקות מובייל תחילה (`flex-col md:flex-row`).
   - ודא תמיכת RTL מלאה (`dir="rtl"` ותגיות התאמה).
   - הוסף JavaScript בסיסי ופעיל (למשל, תפריט המבורגר נפתח עם class toggle).
