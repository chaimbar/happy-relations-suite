## חיבור Supabase משלך לפרויקט

### המצב כרגע
הפרויקט שלך כבר משתמש ב-**Lovable Cloud** (שזה Supabase מנוהל מאחורי הקלעים). אתה רוצה להחליף אותו ב-**חשבון Supabase האישי שלך** (eli0548542222@gmail.com) כדי שתהיה לך שליטה מלאה ויכולת להוציא את הפרויקט מ-Lovable.

### חשוב לדעת
**אני (Lovable AI) לא יכול לבצע את החיבור בעצמי.** החיבור ל-Supabase חיצוני דורש OAuth — אתה חייב להתחבר בעצמך דרך הדפדפן ולאשר הרשאות. זה צעד אבטחה של Supabase ולא ניתן לעקיפה מהצד שלי.

### מה אני כן יכול לעשות אחרי שתחבר
1. להעביר את כל הקוד להשתמש ב-Supabase החדש
2. להעתיק את סכמת הדאטהבייס (טבלאות, RLS, פונקציות) ל-Supabase שלך
3. לעדכן את משתני הסביבה
4. לבדוק שהכל עובד

### איך מחברים — שלבים מדויקים

**אופציה 1: דרך תפריט Cloud (מה שניסינו)**
1. בעורך, לחץ על אייקון **More** (שלוש נקודות / שכבות) בסרגל העליון
2. בחר **Cloud**
3. בתוך מסך Cloud, חפש בפינה הימנית העליונה כפתור **Settings** ⚙️ או **"Disconnect Lovable Cloud"**
4. אחרי ניתוק Lovable Cloud, יופיע כפתור **"Connect Supabase"**

**אופציה 2: דרך Integrations**
1. לחץ על שם הפרויקט בפינה השמאלית העליונה
2. בחר **Settings**
3. בתפריט הצד בחר **Integrations**
4. חפש את **Supabase** ולחץ **Connect**

**אופציה 3: חיפוש מהיר**
- לחץ `Cmd+K` (Mac) או `Ctrl+K` (Windows)
- הקלד "Supabase" — וזה יציע לך ישירות את האפשרות לחיבור
